## ✨ 特色功能

🌏 **多時區支援**

- 台灣時間（預設）
- UTC 世界標準時間

🎨 **彈性化顯示**

- 支援 12/24 小時制切換
- 民國年/西元年顯示選擇
- 臺灣節慶自動顯示

📱 **響應式設計**

- 支援電腦、平板、手機
- 自適應字體大小

🛠️ **工業風格**

- 簡約現代的設計風格
- LED 風格顯示

🔗 **快速分享**

- URL 參數快速設定
- 一鍵複製分享連結

## 🎯 使用場景

- 👨‍💻 工程師的桌面時鐘
- 🏢 辦公室電子顯示屏
- 📱 桌面小工具

## 🚀 快速開始

### 🔧 URL 參數設定

支援透過 URL 參數快速設定時鐘顯示：

```
# 預設設定
index.html

# UTC時間
index.html?tz=utc

# 12小時制
index.html?fmt=12

# 西元年
index.html?year=ad

# 組合使用
index.html?fmt=12&year=ad
```
